## Description

A Pokemon backend application with [NestJS](https://nestjs.com/) using :

- Swagger : [Api Docs v1.0](https://localhost:3000/api/#/)

## Installation

Clone the repository :
`git clone https://github.com/mcholl85/Pokemon-NestJS.git`

Switch to the repo folder :
`cd Pokemon-NestJS`

Install dependencies :
`npm  install`

## Running the app

```bash

# development

npm  run  start



# watch mode

npm  run  start:dev



# production mode

npm  run  start:prod

```

## Test

```bash

# unit tests

npm  run  test



# e2e tests

npm  run  test:e2e



# test coverage

npm  run  test:cov

```
